![Redis](https://img.shields.io/badge/redis-caching-red.svg)

**MailFlow Pro** — это мощная платформа для автоматизации маркетинговых кампаний. Сервис позволяет планировать рассылки, управлять клиентскими базами и отслеживать эффективность каждой кампании в режиме реального времени через интерактивный Dashboard.

## Ключевой функционал
* **Автоматизация по расписанию:** Интеграция с Celery и Redis для выполнения фоновых задач без блокировки основного потока.
* **Умный Dashboard:** Аналитическая панель для мониторинга общего количества рассылок, активных кампаний и уникальных клиентов.
* **Гибкое управление клиентами:** Полноценный CRM-модуль для сегментации получателей.
* **Система логирования:** Детальный трекинг каждой попытки отправки (статусы, ответы почтовых серверов).
* **Безопасность и роли:** Разграничение прав доступа (Manager, Owner, User) с возможностью модерации контента.

## Технологический стек
* **Backend:** Python 3.x, Django (MTV architecture).
* **Database:** PostgreSQL (надежное хранение транзакционных данных).
* **Task Queue:** Celery + Redis (обработка очередей и планировщик задач).
* **Caching:** Redis (оптимизация нагрузки на БД через кэширование представлений).
* **Frontend:** Django Templates + Bootstrap 5 (адаптивная верстка).

## Архитектура решения
Проект спроектирован с соблюдением принципов DRY и SOLID:
* mailings/: Ядро системы. Содержит бизнес-логику рассылок, модели данных и сервисную логику отправки.
* users/: Модуль аутентификации и авторизации с расширенной моделью пользователя.
* management/: Набор кастомных CLI-команд для административного управления сервисом.

## Установка и запуск

1. Клонируйте репозиторий:

    git clone https://github.com/AJLbN0H/mailflow-automation-service.git

2. Настройте окружение:
Создайте файл .env на основе .env.example и заполните данные для PostgreSQL, Redis и SMTP.

3. Запустите проект через Docker (рекомендуется):

    docker-compose up --build

4. Локальный запуск (без Docker):

    poetry install
    python manage.py migrate
    python manage.py runserver

## Тестирование
Для запуска тестов используйте:

    pytest
